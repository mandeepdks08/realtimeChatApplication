# REALTIME CHAT APPLICATION

## Description

The web application titled "REALTIME CHAT APPLICATION" helps two or more people connect and chat on the web in real-time. The people can either do private chatting or they may randomly hop into a public chatroom. The private chats are secure as they are not stored in our database. Also, no one can enter your private chatroom until and unless you share the room ID with him/her. All the members are notified whenever a user leaves or enters a chatroom. Read futher to know how to install this project in your local system.

## Installation

First, install Node.js in your system. You can install it from this [link](https://nodejs.org/en/).

Now, run the following commands in your terminal

```bash
git clone https://github.com/mandeepdks08/realtimeChatApp.git
```

```bash
npm install
```

To start a local server, run

```bash
node server.js
```

Now, you can access the chat application on your browser by typing http://localhost:3000/ in the address bar.
